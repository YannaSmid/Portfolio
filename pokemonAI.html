Gotta cAItch 'em all!

A co-creative system to discover new Pokémon.

Team Members: Nikki Rademaker, Christina Irakleous, Yanna Smid

Project Overview

The Pokémon franchise has captivated fans worldwide with its imaginative designs and iconic creatures. Inspired by this legacy, our team developed a co-creative system that enables users to collaborate with AI to design their own Pokémon. By combining user input with generative AI, the Pokémon Generator turns creativity into reality, blending nostalgia with cutting-edge technology.

For an assignment in Computational Creativity, we were challenged to build a co-creative system. With our interest in Video Games, and nostalgia for Pokemon, we decided to create a co-creative interface that can be used to design new pokémon. Users provide input on their preferred Pokémon (e.g., type, body form, inspiration). Then AI generates a descriptive text and a visual representation of the Pokémon based on the input. The process offers interactive and imaginative design experiences.

Features

  • Interactive Design: Users can select the Pokémon type(s), body shape, and additional requirements via a simple interface.
  • Generated description: AI generates a detailed textual description of the Pokémon using GPT-3.5-Turbo
  • Generated visuals: DALL-E 3 transforms the description into a high-quality Pokémon image.

Technical Details

The system collects user preferences through the interface built in Python Tkinter. Then GPT-3.5-Turbo produces a Pokémon description based on the input, and DALL-E 3 creates a corresponding visual representation. The generated Pokémon description and image are presented together on the interface screen.

  • Python and Tkinter are used for the interface and application.
  • The OpenAI API is used for text and image generation.

Challenges

  • A right prompt had to be found to let the OpenAI create Pokémon of high quality. With some Pokémon types such as Dragon and Fairy type, the created images were more fitting for fantasy fiction rather than in the world of anime.
  • However, the variability in output quality led to iterative improvements in the system prompts. In the end, we found a prompt most suitable for the task of a Pokémon generator.

Process

  • Conceptualization: In the process of coming up with a topuc for a co-creative system we quickly applied our interest in Video Game creation and our interest for a tool to design new Pokémon.
  • Sketches: We started making sketches of a good interface, keeping it as simple and straightforward as possible.
  • Programming: We then implemented the interface and the generative backend for text and visuals.
  • Optimization: Through multiple test cases we were able to refine prompts for GPT-3.5-Turbo and DALL-E 3, to receive images most suitable to the user’s description
  • Evaluation: We let users test the final interface to investigate the creativity of the system based on Standardized Procedure for Evaluating Creative Systems (SPECS) of Jordanous, and let them complete a Turing test.

Reflection

Reflecting on this project, we feel honored to have gotten the opportunity to combine our passion for programming and AI with our interest in Pokémon and Video Games. The Pokemon Generator shows the power of AI in creative domains, enabling users to participate in designing entirely new designs of Pokémon. By merging user preferences with generative AI, the project opened doors to limitless creative possibilities for personal use.
