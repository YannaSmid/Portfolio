Crispr Game9

A playful and creative expression of Crispr Cas9.

By Yanna Smid and Nikki Rademaker

Project Overview

Dive into the fascinating world of genetics and biochemistry with the CRISPR Cas9 Game. This game is a fun and educational mini-game designed to visualize the groundbreaking research on CRISPR-Cas9, published in 2012. This project transforms complex biological concepts into a playful and interactive experience that makes learning about adaptive bacterial immunity both accessible and enjoyable.

The CRISPR Cas9 Game was developed as part of the Playful & Creative Science course, where we were tasked to express the results of a research in a creative manner. My teammate Nikki Rademaker and I chose the article on CRISPR Cas9 because of our background in Biology. We decided to gamify this scientific research and make it understandable for a wider audience. By focusing on the core principles of CRISPR-Cas9, the game aims to simplify a highly technical topic and showcase how gamification can bridge the gap between academic research and everyday learning.

Features

  • Educational Gameplay: Players interact with fundamental CRISPR-Cas9 concepts, including crRNA-DNA pairing, PAM sequence recognition, and the cutting of DNA strands.
  • Two-Part Game Structure to prevent cognitive overload. The first part is a preparation phase where players create complementary crRNA strands and select a Cas9 enzyme. The second part is the main gameplay phase where players locate the correct DNA strand and PAM sequence.
  • Visual Simplicity: Clean visuals and intuitive mechanics ensure the focus remains on learning, avoiding unnecessary distractions.
  • Randomization: DNA strands and PAM sequences are randomly generated in each playthrough, adding replay value and reinforcing core principles.
  • Immediate Feedback: Players see the results of their actions in real time, whether successfully cutting DNA or failing at the wrong PAM site. They are provided with an explanation so they can learn from the mistakes.

Technical Details

  • Programming is done in Unity
  • Interface design is created using Procreate.

Process

  • Research Translation: We first analyzed the 2012 CRISPR-Cas9 study to extract core concepts. Then we were able to simplified these into actionable gameplay elements.
  • Concept Development: We started designing and sketching a game framework that gamifies academic findings while maintaining scientific integrity.
  • Game Creation: We developed the DNA generation system and PAM recognition mechanics, and built the game environment.
  • Testing & Refinement: We conducted user tests to balance educational content with engaging gameplay. This made it possible to adjust visuals, features, and mechanics based on their feedback.
  • Deployment: We published the game on Itch.io for accessibility and feedback.

Reflection

Developing the CRISPR Cas9 Game was an eye-opening experience in how gamification can be used to simplify and disseminate complex scientific ideas. By focusing on educational clarity, we created an engaging tool that helps players understand the CRISPR-Cas9 mechanism without requiring prior knowledge. Balancing gameplay and scientific accuracy was a challenge. While adding elements like enemies or advanced features could have made the game more entertaining, we prioritized simplicity to ensure the educational goal remained clear. User feedback was overwhelmingly positive, with many noting how the game helped them visualize the science behind CRISPR-Cas9.

This project deepened my skills in game design, research translation, and the application of gamification to science communication. It also highlighted the potential of games to make academic research more accessible and engaging.
